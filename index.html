Inlineedit Documentation

Inlineedit is a jQuery plugin that provides a nice looking way to edit and read forms, or select a option. The goal for Inlineedit is to provide a simple yet dynamic tool that makes it very easy to provide users an attractive read or edit. This documentation will provide some examples of how to use Inlineedit. Usage of Inlineedit can be very simple, but for those willing to work a little harder can also provide more sophisticated functionality. The examples aim to provide a good basic idea of how various features can be used without overcomplicating things.

General Use

The first thing to do is to include the javascript files for jQuery and Inlineedit, as well as the Inlineedit stylesheet. These should ideally be included in the head of your html file.

HTML<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<script type="text/javascript" src="jquery.inline-edit.js"></script>
<link href="jquery.inline-edit.css" rel="stylesheet">

Creating a layout

Now, we need to add basic Inlineedit layout to our page

HTML<div class="inline-edit">
  <input type="text" class="form-control input-sm" />
  <a href="#" class="edit"></a>
  <a href="#" class="save"></a>
  <a href="#" class="cancel"></a>
  <span class="chrysanthemum"></span>
</div>

Initialize

When the neccesary files have been included, it's very simple to set up Inlineedit on your page, simply target the element you wish to use with jQuery, and execute the Inlineedit function on it.

Javascript$(".inline-edit").inlineedit();
